## Режим Импорт (код 40) — ПЛАН

### Источник правил
`.cursor/rules/gtd-import-40.mdc`

### Ключевые отличия от Экспорта:

| Аспект | Экспорт (10) | Импорт (40) |
|--------|--------------|-------------|
| Направление | ЭК | ИМ |
| Графа 2 | Резидент (отправитель) | Нерезидент (отправитель) |
| Графа 8 | Нерезидент (получатель) | Резидент (получатель) |
| Графа 15/15a | UZ (readonly) | Страна отправления (ввод) |
| Графа 17/17a | Страна назначения | НЕ заполняется |
| Графа 21 | Отключена | Заполняется |
| Графа 31 | 5 пунктов | 11 пунктов (+ 6-11 для импорта) |
| Графа 36 | Отключена | Заполняется (преференции) |
| Графа 40 | ГТД, ККДГ, МДП | + ТД (транзит), МВ (обязательство) |
| Графа 43 | 0/1 производство | 0/1 собственные нужды |
| Графа 44 | 702 СПТЭК | 701 СПТИМ, 620/629 |
| Графа 46 | По там. стоимости | По фактурной, CIP/CIF |
| Графа 47 | 0% пошлина, 0% НДС | Реальные ставки |
| Графа «С» | 1 поле | 2 поля (+ дата ввоза) |

### Отключённые графы для импорта:
```
4, 6, 10, 16, 27, 51, 52, 53
+ 17/17a через спец. условие в UI
```
