# GTD UZ - Система электронного декларирования

## Описание проекта

Веб-приложение для заполнения и подачи грузовых таможенных деклараций (ГТД) в Республике Узбекистан.

## Основные цели

1. Автоматизация заполнения ГТД
2. Валидация данных согласно официальной инструкции
3. AI-помощник для извлечения данных из документов
4. Генерация PDF деклараций
5. Интеграция с государственными системами

## Правовая основа

- Постановление ГТК РУз от 29.02.2016 №01-02/15-07
- Инструкция о порядке заполнения грузовой таможенной декларации
- Зарегистрировано МЮ 06.04.2016 г. №2773 (с изменениями)

## Таможенные режимы (16 типов)

| Код | Режим |
|-----|-------|
| 10 | Экспорт |
| 11 | Реэкспорт |
| 31 | Временный ввоз (вывоз) |
| 40 | Импорт (выпуск для свободного обращения) |
| 41 | Реимпорт |
| 51 | Переработка на таможенной территории |
| 61 | Переработка вне таможенной территории |
| 70 | Временное хранение |
| 71 | Свободная таможенная зона |
| 72 | Магазин беспошлинной торговли |
| 73 | Свободный склад |
| 74 | Таможенный склад |
| 75 | Отказ в пользу государства |
| 76 | Уничтожение |
| 80 | Транзит |

## Технологический стек

- **Frontend**: Next.js 14, React, TypeScript
- **UI**: Tailwind CSS, shadcn/ui
- **База данных**: Prisma + Supabase (PostgreSQL)
- **AI**: Anthropic Claude API
- **Аутентификация**: NextAuth.js

## Ключевые особенности

1. **Мультирежимность**: каждый из 16 режимов имеет свои правила заполнения граф
2. **Валидация**: проверка данных согласно официальной инструкции
3. **AI-автозаполнение**: извлечение данных из загруженных документов
4. **Генерация PDF**: создание деклараций по официальному шаблону
